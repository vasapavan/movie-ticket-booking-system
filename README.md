A full-stack Movie Ticket Booking System that allows users to browse movies, select shows, choose seats, and book tickets, while providing an admin panel to manage theatres, screens, and show schedules.
-----------------------------------------------------------
features of customers:
View available movies,
Select theatre and show timings,
Choose seats visually,
Book tickets,
View booking confirmation.
-------------------------------------------------
Features of admin;
Add/manage movies
Add theatres and screens
Assign shows to screens
Manage show timings
---------------------------------------------------------
Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python (Flask)
Database: PostgreSQL
-------------------------------------------------
*a sql file is included with which you can create a database structure with sample data.
